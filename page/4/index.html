ubuntu 桌面死机(重启gnome-shell)

发表于 2021-04-19 更新于 2021-04-20 分类于 linux ， ubuntu Valine：
本文字数： 1.7k 阅读时长 ≈ 2 分钟

ubuntu有时桌面会卡死，鼠标点击失灵，解决方法：重启gnome-shell。
建议以下两种二选一：

  • alt + F2 弹出的输入框里输入小写r ，回车，如此重启gnome-shell桌面环境。
  • ctrl + alt + F3 切换到其它虚拟终端tty3(Ubuntu18.04 默认tty1是选择用户登录界面，tty2往往就是对应卡死的桌面位置，我们选择tty3进行操作)，输入用户密码登录，输入ps -ef|grep gnome-shell ，找到tty2对应的gnome-shell pid（一般有两个，一个是tty1的，一个是tty2的），kill -9 pidNum ，系统自动重启gnome-shell,最后crl + alt +F2 切换至tty2,这种方法的优点是不会干扰其他正在运行的程序。
